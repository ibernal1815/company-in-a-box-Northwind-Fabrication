# Environment Overview

## Purpose
This lab environment is designed to model day to day IT operations for a fictional company with about fifty employees across IT, HR, Finance, Engineering, and Sales.

## Host constraints
The environment runs on a single workstation using VMware Workstation Pro.
- Storage: 512GB NVMe
- Memory: 48GB RAM
- CPU: Intel i5 14400F

Design decisions prioritize stability, resource efficiency, and clarity of operation.

## High level components
- Active Directory Domain Services for identity and access management
- Group based access control for file services and administrative permissions
- Basic network segmentation suitable for a small environment
- Centralized logging for servers and a sample of endpoints
- Backups with documented restore testing
