# Internal IT Environment Lab

This repository documents a fictional mid sized company environment built in VMware Workstation Pro on a single workstation. The goal is to demonstrate practical IT operations with realistic constraints, clear documentation, and repeatable procedures.

## What this repo contains
- Documentation for design decisions and build steps
- PowerShell scripts for onboarding and offboarding
- Configuration exports and references
- Evidence from validation and recovery testing

## Scope
Identity and access management, endpoint policy basics, file services, logging, backups, and recovery testing.
